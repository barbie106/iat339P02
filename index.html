P01 - Part 2

Create a style guide that illustrates your brand direction and for sample style guides, visit the course Canvas site.
Your style guide must include names for, examples of and code for the following items:

  • button elements on your site
  • form elements on your site
  • textual elements on your site
  • image elements on your site
  • key groupings of elements (i.e. a navigation bar)

    • Button Elements

    CSS selector:

    'nav a'

    Sample code:

    <a href="#" >Book an appointment</a>

    Rendered element:

    Book an appintment

    Form Elements

    CSS selector:

    'form'

    Sample code:

    <section id="comments" >
    <form method="post" action ="#">

    <div class="input-wrapper" >

    <textarea id="form-comment" placeholder="Please place comment here" >

    </textarea>

    </div>

    <input type="submit" value="Submit your comment" >

    </form>

    Rendered element:

    Texture Elements

    CSS selector:

    'booked'

    Sample code:

    <div class="booked">
    <h3>Appointment is booked</h3>

    Rendered element:

    Appointment is booked!
